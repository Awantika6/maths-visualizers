Data Dimensions

Understanding Scalars, Vectors, Matrices, and Tensors

Scalar

Rank-0

24.5

A single magnitude. No direction.

Vector

Rank-1

[4, -1, 0]

A 1D array. Defines magnitude and direction.

Matrix

Rank-2

[[1, 2], [3, 4]]

A 2D grid. Used for transformations.

Tensor

Rank-n

[[[a,b], [c,d]], [[e,f], [g,h]]]

The multidimensional generalization.

Assessment Tasks

Task 1: The Physics Problem

A car is driving North at 60 mph. In physics, we represent this as (0, 60). What structure is this?

Explanation: Because it has both magnitude and direction, it is a Vector (Rank-1 Tensor).

Task 2: Digital Imagery

A grayscale digital photo is a grid of 1920x1080 brightness values. What structure is this?

Explanation: A 2D grid of numbers organized in rows and columns is a Matrix (Rank-2 Tensor).

Task 3: The Thermometer

The current temperature in this room is exactly 72°F. What structure is this?

Explanation: Temperature is a single numerical value without direction, making it a Scalar (Rank-0 Tensor).
