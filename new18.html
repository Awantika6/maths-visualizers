Singular Value Decomposition (SVD)

CONCEPT

Singular Value Decomposition factorizes any matrix A into U\Sigma Vᵀ, where U and V are orthogonal rotations and Sigma is a diagonal scaling matrix.

U Matrix

Left Singular Vectors. This handles the Final Rotation of the data in the output space.

Σ (Sigma) Matrix

The Scaling Factor. Diagonal values that tell us the "strength" or importance of each dimension.

Vᵀ Matrix

Right Singular Vectors. This handles the Initial Rotation to align the data with principal axes.
